# MetaPiping/MetaStructure 2026.1

## *Launch date*

* June 2026

## *Main features*

>Solver code_aster

* Upgrade to code_aster 17.4 (2025)

>Code updates in ASTER solver :

* B31.3 Ed. 2020 and 2022 (with B31J SIF 2017) and Ed. 2024 (with B31J SIF 2023)

>Analysis

* Multiple modal analysis
* Several RCC-MRx new options

>Structure

* Grid edition
* Allowable stress + compression max in FEA of anchor plates

>Python improvement

* Upgrade to Python 3.10
* Remove of chatGPT libraries

>Design

* Matrix element in MetaStructure
* Move nodes by translation, rotation and mirror

## *Secondary features*

>Loads

* Tree of loads : columns for combination loads with cases and factors

>Analysis

* Print button
* Transient results window

>Materials

* RCC-MRx material inspector + calculator

>Project

* Export project with choice of studies
* Export *.fre for piping models
* Export *.stl

>Supervisor

* Batch calculation also for structures

>Settings

* Preview of DOCX document
  
>Miscellaneous :

* Direct access to current help topic
* 4 modes of superposition check : all pipes, outer pipes transparent, only outer pipe, only inner pipe
* 4 modes of node name : none, all nodes, only restrained nodes, only tee nodes
* New word template for Y vertical
* Anchor plate meshing improvement
* Square selection with ALT pressed (usefull for rigid, spring and matrix or in wireframe mode)
