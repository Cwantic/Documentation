# MetaPiping 2023.1 (Current release)

* May 2023

* Spec driven mode

* POSTR brick in PIPESTRESS plugin
* PIPSYS plugin
* Copy data from review, load and analysis to clipboard
* Screenshot to clipboard
* Distance tool in review
* Python script user manual
